⚙️ Einmalige Einrichtung

Damit alle 4 Personen dieselben Daten sehen, braucht die App eine kostenlose Supabase-Datenbank. Das dauert ca. 5 Minuten.

Schritt 1: Öffne supabase.com → Kostenlos registrieren → Neues Projekt erstellen
Name: rasa-og | Region: EU (Frankfurt)
Schritt 2: Im Supabase-Projekt → SQL Editor → Neues SQL ausführen:
CREATE TABLE rezepte (id text PRIMARY KEY, daten jsonb, geaendert bigint);
ALTER TABLE rezepte DISABLE ROW LEVEL SECURITY;
Schritt 3: Im Supabase-Projekt → Settings → API
→ Project URL und anon public key kopieren
